Programação Web

Na primeira aula prática de Programação Web iremos resolver a primeira ficha de Laboratório(Laboratório 1) abordaremos os princípios básicos associados ao desenvolvimento de uma página Web.

Aprenderão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, farão semanalmente um laboratório, na aula prática (de entrega obrigatória) de aplicação da matéria lecionada, e irão implementar um projeto completo.
